\section{Montagem do Projeto}
\textbf{Planeje}: para que seu projeto saia do papel é preciso planejar tudo que você precisará fazer para executá-lo. Isso significa determinar um horário para o estudo, criar um cronograma, especificar como fará o \textbf{Projeto Integrador}. O ideal é que planeje todos os detalhes para que suas ações sejam mais efetivas.

Este projeto contou com os seguintes materiais abaixo descritos:
\begin{itemize}
	\item 1 LED de alto brilho
	\item 1 Resistor de 220 $\Omega$
	\item 1 Suporte para 2 pilhas
	\item 1 paliteiro
	\item 2 fios (nas cores preta e vermelha) com tamanho suficiente para realizar as ligações, uns 30cm.
	\item Ferro de solda e estanho para as ligações
	\item Fita isolante, fio térmico ou cola quente para isolar as ligações
\end{itemize}
